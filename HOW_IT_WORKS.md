# צ'אטבוט HR — איך זה עובד

## מה זה?

ממשק צ'אט שמאפשר לשאול שאלות בעברית על נתוני כוח אדם ולקבל תשובות ישירות מהדאטאבייס. בלי דשבורדים, בלי פילטרים — פשוט שואלים ומקבלים טבלה עם תשובה.

---

## הזרימה

כל שאלה עוברת 3 שלבים:

**שלב 1 — התאמה מטושטשת (Fuzzy Matching)**
המשתמש כותב בשפה חופשית ("כמה נוכחים בתקשוב?"). המערכת לוקחת את השאלה ומשווה אותה מול כל הערכים שקיימים בדאטאבייס — יחידות, סטטוסים, וסוגי שירות. אם מילה מהדאטאבייס מופיעה בתוך השאלה (למשל "נוכח" בתוך "נוכחים"), היא מקבלת ציון גבוה. ככה ה-AI תמיד עובד עם ערכים מדויקים ולא מנחש.

**שלב 2 — כתיבת שאילתה**
מודל AI מקבל את הערכים המדויקים מהשלב הקודם + את מבנה הטבלה, וכותב שאילתת SQL מתאימה.

**שלב 3 — הרצה והצגה**
השאילתה רצה על הדאטאבייס (Supabase/PostgreSQL), והתוצאות מוצגות כטבלה נקייה בצ'אט, יחד עם ה-SQL שהורץ.

---

## מבנה הפרויקט

| קובץ | תפקיד |
|------|--------|
| `app.py` | ממשק הצ'אט |
| `agent.py` | ה-AI — פרומפט, מודל, חיבור לכלים |
| `tools.py` | שני הכלים: fuzzy matcher + מריץ SQL |
| `db.py` | חיבור לדאטאבייס |
| `.env` | מפתחות API וסיסמת DB |

---

## הדאטאבייס

טבלה אחת: `public.reports`. כל שורה = חייל אחד ביום אחד. מכילה: היררכיית יחידות, דרגה, סוג שירות, סטטוס נוכחות ועוד.
